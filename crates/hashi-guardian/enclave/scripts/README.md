# Enclave Management Scripts

Quick reference for all scripts in this directory.

## Setup & Configuration
- **configure_enclave.sh** - Launch AWS EC2 with Nitro Enclaves
- **user-data.sh** - EC2 user data script (generated by configure)

## Runtime Management
- **expose_enclave.sh** - Expose enclave on port 3000 (VSOCK→TCP)
- **reset_enclave.sh** - Stop enclave & cleanup
- **register_enclave.sh** - Register enclave with attestation

## Development
- **update.sh** - Update StageX image digests
- **changed-files.sh** - List changed files
- **license_check.sh** - Check license headers

## Usage Examples

### Local Development
```bash
# Build and test locally
cd ../src/server && cargo run
cd ../src/client && cargo run
```

### AWS Deployment
```bash
# Configure instance
export KEY_PAIR=my-key-pair
./configure_enclave.sh

# On EC2 instance after cloning repo:
make                    # Build enclave
make run                # Start enclave
./scripts/expose_enclave.sh    # Expose to network
```

### Troubleshooting
```bash
./scripts/reset_enclave.sh      # Reset everything
make run-debug          # Run with debug logs
```
