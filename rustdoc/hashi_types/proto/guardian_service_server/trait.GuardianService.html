where Self: 'async_trait, 'life0: 'async_trait,

Ceremony mode only: rotate the KP set holding the existing BTC key. The operator submits the current KPs’ encrypted old shares in one request; once threshold-many verify, the enclave reconstructs the BTC key, re-splits it for the new KP set, writes the new share state (hashes) to ceremony/, and returns the new encrypted shares.
