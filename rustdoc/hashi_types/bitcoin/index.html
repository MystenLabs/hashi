Module bitcoin

Expand description

Bitcoin primitives shared by Hashi and Guardian. utxo holds the UTXO / transaction types and signing; taproot holds the 2-of-2 descriptor, address, and child-key derivation. Shared address/weight helpers, secp context, and derivation-path alias live here.

Full input weight (WU) for a 2-of-2 taproot script-path spend. TXIN_BASE_WEIGHT (164 WU) + satisfaction (234 WU) = 398 WU (100 vB).
TX_FIXED_WEIGHT_WU
Non-witness fixed overhead for a segwit transaction: nVersion(4x4) + nLockTime(4x4) = 32 WU, plus the segwit marker/flag (2 WU).
