pub fn k256_sk_to_btc_keypair(sk: &SecretKey) -> Keypair
Expand description

Convert a k256::SecretKey to a bitcoin::secp256k1::Keypair. Both libs use big-endian 32-byte scalars so the byte round-trip is value-preserving. We juggle between the two libraries because secp256k1 does not expose the arithmetic tools needed for secret-sharing.
